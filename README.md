# 🎲 Ludo Game (2 Players)

This project is a digital implementation of the classic Ludo board game designed specifically for two players. It provides a simple and interactive way to play Ludo with turn-based gameplay and standard game rules.

## 📌 Features
- Two-player gameplay
- Dice rolling functionality
- Turn-based player movement
- Rule-based token movement
- Token capture system
- Win detection

## 🛠️ Technologies Used
- Python
- Pygame
- Random module

## 🚀 How It Works
1. Two players take turns to roll the dice
2. A player must roll a 6 to start moving tokens
3. Tokens move according to dice value
4. Players can capture opponent tokens
5. First player to move all tokens to the home area wins
